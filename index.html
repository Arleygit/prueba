PASOS PARA CREAR REPOSITORIOS EN GITHUB

1. Abrir cuenta creada en github y nos logeamos, o creamos nuetra cuenta "sign up"

2.Ahora creamos nuestro repositorio en github y lo nombramos

3. Abrimos nuestro repositorio creado en README.md

4. para adjuntar o descargar archivos en el repositorio de github lo hacemos mediante la aplicacion gitbush

git init: es para inicializar la conexion como repositorio (activando el modo master de nuestro repositorio)

git clone: para copiar el repositorio creado en github en nuestro work space

"crear archivos con comandos linux con los comandos vi o nano segun su preferencia y creamos un archivo tipo archivo.txt"

git add: adicionar cualquier cambio en el archivo

git commit -m 'comentario': comentar nuestro archivo que se ha de subir

git pull: permite el fusionar nuestros repositorios

git remote add origin Url: permite identificar el repositorio a donde se desea subir el archivo

git push: para empujar el archivo actualizado a github (repositorio local)
